# To_Do_List-App
<br><br>

## This is a To-Do List App Project

<br>

### The technologies we are going to use are: 
<br>
#### <b>Frontend: </b> HTML, Bootstrap, Jinja2 (For Templating) <br>
#### <b>Backend: </b> Flask, Flask-WTF <br>
#### <b>Database: </b> SQLite3, SQLAlchemy <br>

## Core feature:

<ul>
<li> User login
<li> User logout
<li> Create Task
<li> View Task
<li> Update Task
</ul>
